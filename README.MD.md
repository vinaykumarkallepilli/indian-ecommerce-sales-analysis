# Business Problem

The company wants to monitor:
- Sales Performance
- Profitability
- Target Achievement
- Customer Performance
- Product Performance
- Geographic Performance

using one interactive Power BI dashboard.

---

# Solution

This project cleans, models and analyzes the data using SQL and Power BI.

The final solution includes
- SQL Analysis
- Power Query Transformation
- DAX KPIs
- Interactive Dashboard
- Business Insights

# Key Business Insights

- Electronics generated the highest revenue.
- Clothing achieved the highest profit.
- Sales performance was analyzed against monthly targets.
- Top customers contributed a significant share of total revenue.
- Geographic analysis identified the highest-performing states.
- Product analysis highlighted top-selling sub-categories.
